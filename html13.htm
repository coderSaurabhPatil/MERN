HTML5 Content Tags

HTML5 introduces new semantic elements to better structure content:

  • <header>: Represents the header section of the page.
  • <section>: Defines a section in the document.
  • <article>: Represents an independent piece of content.
  • <aside>: Represents additional content, like a sidebar.
  • <footer>: Defines the footer of the page.
